● 痛点洞察

每一个 OIer 都经历过的困境

在 Luogu、AtCoder、Codeforces 上刷题时，错题总是零散分布，难以系统化复盘。

❓

题解看不懂

官方题解跳步严重，讨论区解释零散晦涩。别人的思路和自己的代码之间，总隔着一层迷雾。

🤖

AI 不了解上下文

把代码丢给 ChatGPT，它不知道题目背景、测试数据、提交平台。得不到"为什么这写法不对"的精准答案。

🔄

同类错误反复犯

这次数组越界，下次整数溢出，再下次又是边界条件。同一坑踩了一次又一次，没有系统化反馈。

● 核心功能

六大核心能力

每一个功能都围绕一个目标：构建你的学习画像，让数据驱动你的竞赛之路。

📊
核心能力

智能学习画像

分析错题记录和提交时间，自动生成学习画像：

  • 学习时段分布：你是夜猫子还是早起党？
  • 错误类型占比：WA / RE / TLE / CE 哪个最常犯？
  • 薄弱算法领域：动态规划 / 图论 / 数论？
  • 代码风格特点：缩进 / 命名 / 注释习惯
🔍
全 AI 分析

AI 自动生成错误点

粘贴代码后，AI 自动分析并生成结构化的错误检查点列表：变量名一致性、循环边界、输入输出格式等具体检查项，不再依赖经验。

🔗
账号同步

Codeforces API 集成

保存 CF 账号一键获取真实提交记录，自动分析 WA/RE/TLE/CE 等错误提交，融入用户画像。支持 Codeforces、洛谷、AtCoder 等主流平台。

💡
引导式反馈

分层提示系统

低级错误：AI 直接指出问题位置和原因。深层问题：分层递进提示，引导你自己找到答案。锻炼 Debug 能力，而非养成"看答案"的依赖。

🎨
主题切换

亮色 / 暗色主题

支持亮色和暗色两种主题模式，点击右上角按钮即可切换。设置自动保存，下次打开保持上次选择，适配不同使用场景。

🧠
多模型聚合

模型广场 · Auto 模式

内置百度 / DuckDuckGo 反代模型，支持用户添加 OpenAI 兼容 API。Auto 模式会依次探测可用模型，确保在某一服务不可用时仍能继续获得 AI 辅助。

● 技术架构

纯前端 + 本地存储 + 多源 AI

基于 Web 的智能错题管理平台，数据留在本地，AI 分析在云端。集成开源反代项目，让免费 AI 触手可及。

链接解析与抓取 Step 01

用户粘贴提交链接并选择 OJ 平台，系统自动抓取提交页面，提取代码内容和题目信息。

多源 AI 反代调用 Step 02

本地部署 baidu-2api 与 DDG2API，无需 API Key 即可免费调用百度文心一言与 DuckDuckGo AI。

AI 错误分析 Step 03

AI 自动生成错误分类和错误点清单，结合用户历史错题进行模式识别。

本地数据存储 Step 04

使用 localStorage 在用户本地电脑存储所有数据，不上传服务器，保护隐私安全。

用户画像生成 Step 05

跨题目分析错误模式，结合真实提交记录，建立个人错误画像，识别薄弱算法领域。

模型池统一管理 Step 06

内置百度 / DuckDuckGo 免费模型 + 用户可添加 OpenAI 兼容 API，Auto 模式自动降级保证可用性。

● 模型与 API 推荐

选对你的 AI 后端，让分析更聪明

WAnote 默认通过 duck.ai 与 chat.baidu.com 的反代提供免费 AI 回答。如果你拥有自己的 API，模型质量与分析深度都会显著更强。以下是几个值得一试的平台：

注意：下方推荐仅供参考，请根据自身需求与预算选择，并注意保护账号与 API Key 安全。
01
便宜 聪明

DeepSeek

deepseek.com ↗

价格便宜还聪明，非常好用。国产模型中性价比首选，适合追求低成本与高质量兼顾的用户。

02
中转站 不溢价

GPTsAPI

gptsapi.net ↗

价格与原厂相同，不会溢价。如果想用一个 API Key 管理多个模型，这个中转站非常合适。

03
可免费 模型多

iamhc

api.iamhc.cn ↗

模型数量比 GPTsAPI 还要多，并且提供免费使用额度。付费档位可能略高于原厂价格，但胜在选择丰富。

04
免费 7B/8B 需实名

SiliconFlow

siliconflow.cn ↗

知名中转站，提供 7B / 8B 规模的免费模型，适合轻度测试与学习。注册时需要实名认证。

如果暂时没有预算，api.iamhc.cn 是一个不错的起点；如果实在无法注册，WAnote 自带的 反代模型 也能满足日常使用。
但我们仍然 非常建议 你购买属于自己的 API —— 更稳定、更聪明、更可控。
